

## Sobre o Jogo

Corra o quanto puder com seu cubo 3d.


## Funcionalidades

- Gráficos 3d
- Mecanicas simples de jogo

## Como Jogar

1. Controle o quadrado com as setas do teclado.
2. pause com a letra P OU ESC.
3. Evite colidir com os objetos.
